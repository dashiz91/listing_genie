# Story 2.3: Generate Infographic Images (2)

**Epic:** Epic 2 - Image Generation Engine
**Story ID:** 2.3
**Title:** Generate Infographic Images (2)
**Priority:** Must Have
**Complexity:** Large
**Status:** Draft

---

## User Story

As a user, I want infographic images that highlight my product's key features so that customers quickly understand my product's benefits.

---

## Acceptance Criteria

- [ ] System generates 2 infographic images
- [ ] Infographic 1 highlights key feature #1 with bold text
- [ ] Infographic 2 highlights key feature #2 with bold text
- [ ] Text is large and readable at thumbnail size
- [ ] Product is visible alongside feature callouts
- [ ] Text density is under 20% of image area
- [ ] Output resolution is 2000x2000px minimum
- [ ] Images saved to cloud storage with unique identifiers

---

## Technical Notes

### Infographic Creative Goals
From `docs/creative-blueprint.md` Section 2:
- **Goal:** Educate quickly with scannable content
- **Technique:** Text as design element, not just caption
- **Rules:** Benefits over features ("Sleep Better" not "10mg Melatonin")

### Prompt Template Reference
See `docs/architecture.md` Section 5.3 for infographic.py templates.

**Infographic 1:**
- Uses feature_1 as key benefit
- Center-left product placement
- F-pattern reading flow

**Infographic 2:**
- Uses feature_2 as key benefit
- Different visual approach to avoid repetition
- Maintains style consistency with Infographic 1

### Micro-Copywriting Rules
| Rule | Description |
|------|-------------|
| Text as Design | Use text overlays as design element |
| Bold & High-Contrast | Legible at thumbnail size |
| Benefits Over Features | "Sleep Better" not "10mg Melatonin" |
| One Message Per Image | Single objective per infographic |

### Storage Paths
- `storage/generated/{session_id}/infographic_1.png`
- `storage/generated/{session_id}/infographic_2.png`
